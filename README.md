# Sistema Digestivo — Trabajo Práctico de Biología

Este es el repositorio de la página web que armamos para la exposición de Biología en **4º C Sociales**. 
La idea fue salir de las típicas láminas de cartulina de siempre y hacer algo digital e interactivo para mostrar en clase.

## 👥 Integrantes
* **Genaro Gularte** (Código y diseño)
* **Valentino Belletti** (Información y contenido)

## 🛠️ ¿Cómo está hecha?
* **HTML5:** Para armar toda la estructura de los textos de cada órgano.
* **CSS3:** Para dejar la página estética, meterle las fuentes lindas y que quede bien ordenada.
* **JavaScript:** Lo usamos para darle interactividad al menú lateral de navegación.

## 📖 Temario incluido
En la página web van a encontrar la información completa y detallada de:
1. Glándulas salivales
2. Boca (Dientes y Lengua)
3. Faringe
4. Laringe y epiglotis
5. Esófago
6. Estómago
7. Hígado
8. Vesícula biliar
9. Páncreas
10. Intestino delgado
11. Intestino grueso
